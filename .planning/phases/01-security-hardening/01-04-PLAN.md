Find the location in `buildApp()` where the application is configured but before `return app`. Add the DATA_DIR assertion immediately after `loadConfig()` is called or after the app object is created with the config:

```typescript
// DATA_DIR safety assertion — refuse to start if session files would land inside the repo
const dataDir = resolve(app.config.DATA_DIR);
const projectRoot = resolve(process.cwd());

if (dataDir.startsWith(projectRoot + '/') || dataDir === projectRoot) {
  app.log.fatal(
    { dataDir, projectRoot },
```
